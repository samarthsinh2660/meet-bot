# prowlermeet-assistant

Skriber / meeting-intelligence landing page built with React, TypeScript, Vite, Tailwind CSS, and shadcn-ui.

## Features

- Autonomous meeting intelligence marketing site
- Modern responsive UI with hero, dashboard preview, and CTAs
- Dark theme by default

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting started

Prerequisites:

- Node.js 18+ and npm

Install dependencies:

```sh
npm install
```

Start the dev server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

Preview the production build locally:

```sh
npm run preview
```

## Project structure

- `src/main.tsx` – React entrypoint
- `src/App.tsx` – root app shell and routing
- `src/pages/` – top-level pages
- `src/components/` – shared UI components

## Deployment

You can deploy the built `dist/` folder to any static hosting provider (for example, Netlify, Vercel, or Cloudflare Pages). Run `npm run build` and point your host at the `dist` directory.
